# Магазин 

## Назначение программы

Приложение предназначено для просмотра товаров и оформления заказов в магазине обуви.

## Подготовка к запуску

Перед запуском программы необходимо:

1. Запустить Open Server.
2. Запустить сервер MySQL.
3. Запустить базу данных .
4. Убедиться, что параметры подключения к базе данных указаны корректно.

## Запуск программы

1. Откройте папку с программой.
2. Запустите  программу.
3. Выполните вход в систему или нажмите кнопку «Войти как гость».

## Пароль и логин для входа в программу 
- Администратор:admin ; admin
- Менеджер:manager ; manager
- Пользователь:user1 ; user1
- Пользователь:user2 ; user2
  
## Роли пользователей

### Гость
- Просмотр каталога.
- Просмотр информации о товарах.

### Пользователь
- Просмотр каталога.
- Просмотр информации о товарах.

### Менеджер
- Просмотр каталога
- Поиск, фильтрация,сортировка,категории.
- Просмотр заказов

### Администратор
- Просмотр каталога
- Поиск, фильтрация,сортировка,категории
- Добавление товара
- Редактирование товара
- Удаление товара
- Управление заказами



## Используемые технологии

- Python
- PyQt6
- MySQL
- PyMySQL
