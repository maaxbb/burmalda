Магазин обуви
Назначение программы
Приложение предназначено для просмотра товаров и оформления заказов в магазине обуви.

Подготовка к запуску
Перед запуском программы необходимо:

Запустить Open Server.
Запустить сервер MySQL.
Импортировать базу данных из файла database.sql.
Убедиться, что параметры подключения к базе данных указаны корректно.
Запуск программы
Откройте папку с программой.
Запустите файл Магазин обуви.exe.
Выполните вход в систему или нажмите кнопку «Войти как гость».
Роли пользователей
Гость
Просмотр товаров.
Поиск товаров.
Сортировка и фильтрация товаров.
Клиент
Просмотр товаров.
Оформление заказов.
Просмотр своих заказов.
Менеджер
Работа с товарами и заказами.
Поиск, фильтрация и сортировка.
Администратор
Добавление, редактирование и удаление товаров.
Управление заказами.
(Чтобы что-то отредактировать, администратору необходимо нажать правой кнопкой мыши. А чтобы что-то удалить, администратору необходимо нажать двойным нажатием кнопки мыши.)
